
Requirements:
Create an app for yoga teachers as the target. The problem they have is for each class they have to create a flow of different asanas with different criteria, usually they do this by hand in a sheet of paper that is labor intensive, not a useful visual aid to learn the class by heart and laborious. The app should solve this problem. 
Key user flows
As a yoga teacher I want to log in to the app and have a repository of asanas that I can curate. The yoga teacher also wants to create a class that consists of a series of asanas.  Asanas are different postures and the app has a directory with the image of the asana and a file with metadata that has name in sanskrit, other names , muscles involved in the asana, series that the asana belongs to , duration , what type of assists can you make, 
Flow1- As a yoga teacher I login and i have an option to create a new class. A class has a duration, level of expertise of the students, could belong to a pre-defined series like ashtanga series 1 or rocket yoga, a type of class a focus.
Flow 2 - Once the class is created I can choose form the repository of asanas the ones I want tp include inmy class, I have filters based in all the metadata of the asanas repo.
flow-3 When I choose an asana i can configure number of repetitions or progressions for an asana, progressions are variations of the same asana. SO I can add 3 different asanas but it will only count as the duration of one. 
Flow 3 - When I am configuring my class I can see the total duration of the class based on the asanas I have chosen.
Flow 4- I can also replay the whole class or selected asanas to see the flow. 
Flow 5- When I am done I can save my class in a catalog of classes that I curate and that are persisted.
Flow 6 - I can also download the whole class as a PDF
Flow 7 - When I select the asana from the catalog I can doubleclick and see the description of the asanas
Flow 8 - I can see in the catalog if an image is verified or not
